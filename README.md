# List of Films

## :man_technologist: Technologies

- Angular 11
- Angular Material

To see the project working, you first need to start json-server.
Open the command line and run the following command `json-server --watch db.json`

With that you can run locally or through this link [aqui](https://list-of-films.netlify.app/movies), which I published on [Netlify](https://www.netlify.com/)
